📚 Book Explorer App
🚀 Overview
Book Explorer is a React-based web application that allows users to search and explore books using the Open Library API. It provides a clean interface to discover books, view details, and handle different UI states like loading and errors.

✨ Features

🔍 Search books by title
📖 View detailed book information
⚡ Fast API integration using Open Library
🎯 Handles loading, error, and empty states
🧠 State management using React Context
🎨 Clean and responsive UI


🛠️ Tech Stack

React.js (Vite)
JavaScript (ES6+)
CSS3
Open Library API


📂 Project Structure
src/
 ├── components/     # Reusable UI components
 ├── pages/          # Main pages (Home, BookDetails)
 ├── context/        # Global state (BookContext)
 ├── services/       # API calls
 ├── utils/          # Helper functions


▶️ How to Run Locally

Clone the repository:

git clone https://github.com/YOUR_USERNAME/book-explorer-app.git


Install dependencies:

npm install


Start development server:

npm run dev


Build for production:

npm run build


Preview production build:

npm run preview
